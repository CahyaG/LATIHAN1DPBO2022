### Usage

#### C++
```
g++ -c *.cpp
g++ main.cpp -o [name].exe
```

#### Java
```
javac *.java
java main
```

#### PHP
- Letakkan Folder yang berisi kode PHP di folder htdocs pada folder XAMPP
- Buka XAMPP dan jalankan Apache Web Server
- Buka Browser dan buka alamat berikut<br>
```localhost/[nama-folder]```

#### Python
```
py [name].py
```
